# Installation

## Stable release

To install geoedemo, run this command in your terminal:

```
pip install geoedemo
```

This is the preferred method to install geoedemo, as it will always install the most recent stable release.

If you don't have [pip](https://pip.pypa.io) installed, this [Python installation guide](http://docs.python-guide.org/en/latest/starting/installation/) can guide you through the process.

## From sources

The sources for geoedemo can be downloaded from the Github repo.

You can clone the public repository:

```
git clone git://github.com/shakurgds/geoedemo
```
